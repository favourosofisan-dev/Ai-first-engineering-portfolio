Case Study

DE GLEE Beauty Ecommerce

A mobile-first, luxury beauty storefront built with plain HTML, CSS, JavaScript, and a lightweight Node.js backend for secure account, product, and order handling.

Beauty & Cosmetics Frontend + Node.js Backend JSON persistence
Live website →

Project Overview

DE GLEE is a mobile-first ecommerce storefront designed for a luxury beauty brand. It showcases curated skincare and body care products while delivering a polished, accessible shopping experience that can be deployed without a build pipeline.

Problem

The client needed a modern beauty storefront that highlights premium products, supports customer login and cart persistence, enables secure owner product updates, and stays lightweight for simple deployment.

Solution

I built a responsive frontend using plain HTML, CSS, and JavaScript, paired with a minimal Node.js backend to support protected owner access, product CRUD, and order collection.

Key Features

  • Modern luxury storefront with hero sections, category browsing, and search.
  • Mobile-first responsive design with product quick-view and recommendation guidance.
  • Cart add/update functionality with persistence via localStorage.
  • Email sign in / sign up flow with cookie-based sessions for customers and admins.
  • Protected /admin dashboard for product management and image uploads.
  • Checkout flow saving orders to data/orders.json and payment instructions via WhatsApp.

Implementation Highlights

  • Built a Node.js HTTP server with no external framework dependency.
  • Seeded product data automatically when products.json was missing.
  • Secured passwords with built-in crypto.scryptSync.
  • Managed sessions in-memory and stored authentication cookies securely.
  • Used JSON file storage for easy deployment and low maintenance.

Business Value

  • Launch-ready beauty storefront with owner-controlled catalog management.
  • Low-maintenance architecture that's easy to deploy and update.
  • Improved customer trust through secure login and polished shopping UX.
  • Flexible order pipeline with owner alerts and transfer instructions.

Future Improvements

  • Add real payment gateway integration such as Stripe or Paystack.
  • Replace demo checkout with live payment and order processing.
  • Add inventory tracking and order status management in admin.
  • Support social login and richer customer profiles.
  • Migrate from JSON storage to a database for scale.
